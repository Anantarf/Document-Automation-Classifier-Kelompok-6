# Frontend - Document Automation

Scaffold created using Vite + React 18 + TypeScript + Tailwind.

Dev:

- Install deps: `npm install`
- Start dev server: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint` (requires dev dependencies installed)
- Test: `npm run test` (vitest)

API base is configured in `.env` (VITE_API_BASE)
